# Personality Lab TH

แบบประเมินบุคลิกภาพภาษาไทย 3 รูปแบบในเว็บเดียว:

- DISC Profile — 30 ข้อ
- MBTI — 32 ข้อจากเอกสารต้นฉบับ
- Enneagram — 81 ข้อ (9 Type × 9 ข้อ)

เว็บทำงานแบบ client-side ทั้งหมด รองรับการบันทึกชื่อผู้ทำแบบประเมินหลายคน โดยแยกคำตอบและผล DISC, MBTI และ Enneagram ของแต่ละคนไว้ใน `localStorage` ของเบราว์เซอร์ พร้อมหน้าสรุปผลทั้ง 3 แบบ ปุ่มคัดลอก และดาวน์โหลดผลเป็นไฟล์ข้อความ ข้อมูลจะไม่ถูกส่งไปยังเซิร์ฟเวอร์

## หมายเหตุ

ใช้เพื่อการสำรวจตนเองเท่านั้น ไม่ใช่เครื่องมือวินิจฉัยทางการแพทย์หรือจิตวิทยา และไม่ควรใช้เป็นเกณฑ์เดียวในการคัดเลือกบุคลากร
